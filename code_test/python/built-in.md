# 最初から使える関数たち



## 変数などが特定の型かどうかを調べる : isinstance関数
```
name = 'タマ'
result = isinstance(name, str)
print(result)
```



## インスタンスの型を取得する : type関数
```
print(type(100))
```
<class 'int'>

```
print(type('猫'))
```
<class 'str'>
