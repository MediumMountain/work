# 設計書などの構成
```
ディレクトリ構成
root/
   ├pdf/ 
   │   ├要件定義/
   │   │     └ 要件定義書.pdf
   │   ├基本設計
   │   │     ├ 画面遷移図.pdf
   │   │     ├ 画面一覧.pdf
   │   │     ├ テーブル定義書.pdf
   │   │     └ 機能一覧.pdf
   │     etc..
   │
   └md/
      ├要件定義/
      │     └ 要件定義書.md
      ├基本設計
      │     ├ 画面遷移図.md
      │     ├ 画面一覧.md
      │     ├ テーブル定義書.md
      │     └ 機能一覧.md
　　    etc..
```
